# hanum

한세인 도우미 앱, 한움입니다.

## commands

```bash
$ git clone https://github.com/hansei-hanum/hanum-frontend.git
$ cd hanum-frontend
$ yarn install

# command of a specific package
$ yarn <workspace-name> <command>

# example
$ yarn hanum-auth ios # run ios app
```
